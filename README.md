
  # Prdaedium Hackathon

  This is a code bundle for Praedium Hackathon. The original project is available at https://www.figma.com/design/eJzR9oFGQWytobkGR0CKq0/FarmOS-Hackathon.

  This project was originally named FarmOS, but Praedium felt like a more fitting name, it is also the Latin Name for "Farm"
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
